# VAI BI

Base inicial del sistema gerencial web para VAI.

## Estructura

- `apps/web`: frontend React + Vite
- `apps/api`: backend Express
- `db`: esquema inicial y notas del modelo analitico
- `docs`: documentacion funcional y tecnica

## Primer alcance

- Directorio
- Comercial
- Base tecnica para sumar Cobranzas, Instalaciones, Soporte y Caja
